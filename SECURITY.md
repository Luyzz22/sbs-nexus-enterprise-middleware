# Security Policy - SBS Deutschland GmbH (SBS Nexus)

## GDPR & Compliance Commitment
Every component within this repository is tailored for deployment in the DACH region and adheres strictly to DSGVO (GDPR) requirements. Data processing is legally bound to German sovereign server structures (Frankfurt am Main).

## Reporting a Vulnerability
Please do not open public GitHub issues for security vulnerabilities. Instead, report any compliance or security risks directly to:
**security@sbsdeutschland.com**

We corporate with Chief Information Security Officers (CISOs) under strict NDA conditions.
